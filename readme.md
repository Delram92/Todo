La aplicación se encuentra disponible en:  https://todo-210302.appspot.com/

Las entidades creadas son las siguietes:

	Todo y TodoEstado

	Se pueden crear tantos estados se requiera, pero no se tiene la opcion de editarlos ni eliminarlos.
	Las tareas se pueden crear, actualizar, eliminar y ver su detalle, no se puede editar la prioridad, la fecha de inicio, ni el nombre de la tarea.


Para subirla localmente:

	1. es necesario tener instalado y configurado google appengine en la maquina. 
	2. Descargar app engine desde la pagina: https://cloud.google.com/appengine/downloads
	3. Instalar cloud sdk. 
	4. Descargar eclipse neon y el plugin de app engine.
	5. Instalar maven en el equipo. 
	6. Una vez instalado y configurado appengine y maven en el equipo, se ubica en la raiz del proyecto
	   y se ejecuta la aplicación con el siguiente comando: mvn appengine:run
